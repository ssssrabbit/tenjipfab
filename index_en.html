Features

3 Core Features

Advanced Translation

Automatically converts Kanji to Kana, handles word spacing (wakachigaki), and distinguishes particle readings. (e.g., "Ha" to "Wa").

Interactive Editing

You don't have to rely solely on automation. Simply tap the Braille blocks on the preview to correct readings instantly.

Optimized for 3D Print

Instantly exports STL data with shapes easy to use in the field, including auto-splitting for long texts, rounded corners, and strap holes.

How to Use

Easy 3-Step Process

1

Input & Convert

Type Japanese text into the app. Check the Braille result in the preview and tap to correct if necessary.

2

Save as ZIP

Tap "Save" to download a ZIP file. It contains the STL (3D data) and an HTML guide for verification.

3

Send to Printer

Load the STL file into your 3D printer and start printing! Your Braille plate will be ready in minutes.
